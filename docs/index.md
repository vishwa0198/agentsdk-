# agentsdk

A lightweight Python SDK for building AI agents with tools, memory, and multi-agent pipelines.

## Features

- ReAct agent loop with parallel tool execution
- `@tool` decorator — JSON Schema auto-generated from type hints
- Multi-agent DAG pipelines and async message bus
- Persistent sessions with checkpoint and fork/resume
- OpenTelemetry tracing built in
- `scaffold-agent` CLI

## Install

```bash
pip install agentsdk
```
